Detailed Description

Implements the source_location struct.

This struct is not supported until c++17 and it is thus not possible to use the standard version. The implementation tries to behave somewhat similar to the standard version so that it should be easier to using it in the future. The name of the struct is the same as the one in the standard library and have thus a different naming convention.

The implementation is based on GCC provided builtin functions. It will always return 0 for the column number as this is not supported by the GCC builtin functions.
