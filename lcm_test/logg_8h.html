Detailed Description

The file defines logging support.

The purpose is to introduce logging support by including only this header file. This makes it easier to change the lower level logging implementation to something else. This file serves as the interface towards the logging support.

The logging is implemented without any macro definitions as the code is written without the use of any macros. This to avoid the problems that might come from the fact that the macros do not have any namespaces but are global text replacements.

The source location is provided by using default parameters that injects the information. These must be placed last in the argument list according to c++ rules. This makes it impossible to use variable number of arguments at the same time. To anyway have support for multiple output streams are used. Several output streams are defined and can be used for a more flexible generation of logging messages.

logg::Info() << "Lisa was also here at " << MAX;
